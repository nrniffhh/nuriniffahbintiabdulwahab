Laboratory Exercises

Lab 1: User Interface (UI)

Introduction to Android layout foundations using declarative UI paradigms in Jetpack Compose, organizing components via Rows, Columns, and Boxes.

Reflection: In Lab 1, I discovered how to orchestrate interface layouts declaratively. Grasping dynamic state alignments was initially tricky, but structured code hierarchy helped solve it quickly.

Lab 2: Interaction

Handling custom user behaviors, button clicks, form inputs, and tracking dynamic mutations via State Management patterns.

Reflection: I learned to implement 'remember' and 'mutableStateOf' mechanics for fluid input processing. This experience made my app significantly more responsive to immediate user inputs.

Lab 3: Material Design

Applying modern Material 3 design directives to configure structural themes, custom typography, and accessible component aesthetics.

Reflection: Material Design integrations completely elevated the visual aesthetics of my application. Aligning design choices to health-themed palettes (SDG 3) yielded a highly professional finish.

Lab 4: Navigation

Constructing multi-screen transitions and data pipeline arguments using standard NavHost and NavController APIs.

Reflection: Passing structured parameters across application views required precise destination modeling. Mastering navigation was vital for streamlining complex, real-world application workflows.

Lab 5: Room Database

Integrating offline-first architecture by configuring persistent database caches utilizing the Room Persistence Library.

Reflection: This practical assignment clarified core aspects of DAOs, Entities, and Repository abstractions. Local caching secures user experiences even under erratic network environments.
